== Description ==

**ZenSEO AI Engine** is a fully automated WordPress SEO plugin powered by artificial intelligence. It analyzes your content, generates optimized meta tags, creates schema markup, and provides actionable tasks - all without human intervention.

### Features

**🔍 SEO Analysis**
- Title tag optimization scoring
- Meta description analysis
- Heading structure evaluation
- Keyword density checking
- Image alt text verification
- Internal/external link analysis
- Schema markup detection
- Readability scoring (Flesch-Kincaid)

**🤖 AI-Powered**
- Auto-optimize content for keywords
- Generate SEO-friendly titles & meta descriptions
- Create fully optimized articles
- Improve content readability
- Build semantic content clusters

**📊 Dashboard**
- Visual content scores
- Priority task list
- Quick action buttons
- Bulk analyze capability

**🔗 Integrations**
- Google Search Console
- Bing Webmaster Tools
- SEMrush API (optional)
- WordPress REST API

**📋 Automated Tasks**
- Auto-generated SEO tasks
- Priority-based recommendations
- Task completion tracking

### Supported AI Providers
- OpenAI (GPT-4)
- OpenRouter (multi-provider)
- Anthropic (Claude)

== Installation ==

1. Upload the `zenseo` folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to ZenSEO > Settings to configure your AI provider API key
4. Start analyzing your content!

== Frequently Asked Questions ==

= Do I need an AI API key? =
Yes, you need an API key from OpenAI, OpenRouter, or Anthropic to use the AI features. The basic SEO analysis works without it.

= How does this compare to Yoast/RankMath? =
ZenSEO does everything those plugins do - plus AI-powered automation. You get the same scoring system but with AI that can actually fix the issues for you.

= Will this slow down my site? =
No. The plugin is lightweight and all AI processing happens asynchronously. Your pages load normally.

= Can I use it with other SEO plugins? =
Yes, but we recommend disabling Yoast/RankMath scoring to avoid confusion. ZenSEO will work alongside any security or caching plugins.
